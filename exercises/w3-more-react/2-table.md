# En tabell!

1. Skapa en komponent som tar ett objekt som en prop och renderar ett `<table>`-element.
2. Tabellen ska ha två kolumner: `Name` och `Age`. 

Objektet ska ha följande form: 
```
const people = [
  {
    name: "Ryan",
    age: 41
  },
  {
    name: "Bryan",
    age: 68
  },
  //...
];
```
