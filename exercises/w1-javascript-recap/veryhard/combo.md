# Letter combo

Skriv en funktion `combo` som tar en sträng som argument. Du ska returnera en array med alla olika kombinationer av bokstäver i strängen. Längden på varje combo ska också vara samma längd som argumentet.

**Exempel**

```
combo("ABC"); // =>
/*
  [
    'AAA', 'AAB', 'AAC', 'ABA',
    'ABB', 'ABC', 'ACA', 'ACB',
    'ACC', 'BAA', 'BAB', 'BAC',
    'BBA', 'BBB', 'BBC', 'BCA',
    'BCB', 'BCC', 'CAA', 'CAB',
    'CAC', 'CBA', 'CBB', 'CBC',
    'CCA', 'CCB', 'CCC'
  ]
*/
```
